# Grafo Exemplo

## Representação da Matriz de Adjacência

|   | A | B | C | D | E | F |
|---|---|---|---|---|---|---|
| **A** | 0 | X | 0 | X | X | 0 |
| **B** | X | 0 | X | X | X | 0 |
| **C** | 0 | X | 0 | X | X | X |
| **D** | X | X | X | 0 | 0 | X |
| **E** | X | X | X | 0 | 0 | X |
| **F** | 0 | 0 | X | X | X | 0 |

### Legenda:
- `0` representa a **ausência de conexão** entre os vértices.
- `X` representa a **presença de uma conexão** entre os vértices.
